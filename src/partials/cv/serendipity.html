Serendipity

Web Developer - UI/UX

August 2020

Serendipity is a UI Theme that I created after moving to MacOs and having vision issues while using VS code with other themes. Right now the theme has 8k users and growing.

Serendipity is a theme prepared for Retina displays with 6 different variations:

  • Dark
  • Dark italic
  • Dark low contrast
  • Light
  • Light Low contrast
  • Light Low contrast
  • iTerms2 Dark
  • iTerms2 Light
  • MacOS terminal Dark
  • Slack Dark
  • Slack Light
  • Linear app Dark
  • Linear app Light
  • Firefox Dark
  • Firefox Light
